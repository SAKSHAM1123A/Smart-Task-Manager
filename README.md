**🚀 Smart Task Manager (AngularJS SPA)**

A modern and responsive Single Page Application (SPA) built using AngularJS (1.8.2) that helps users efficiently manage daily tasks with a simple and interactive UI.

**✨ Features**

➕ Add new tasks instantly
❌ Delete unwanted tasks
✔ Mark tasks as completed
🔍 Live search & filtering of tasks
📊 Task statistics (total & completed)
💾 Data persistence using localStorage
🎨 Clean, responsive, and modern UI
🛠️ Tech Stack
🌐 HTML5
🎨 CSS3
⚙️ JavaScript (ES6)
⚡ AngularJS (1.8.2)

**📁 Project Structure**
project/
│── index.html
│── style.css
│── app.js
▶️ How to Run This Project
Download or clone this repository
Open the folder in VS Code or any editor
Run index.html in your browser

💡 Recommended: Use Live Server extension (VS Code) for best experience

**🧠 AngularJS Concepts Used**

ng-app → Initializes AngularJS application
ng-controller → Connects UI with logic
ng-model → Two-way data binding
ng-repeat → Dynamic rendering of tasks
ng-click → Handles user actions
ng-class → Conditional styling
Filters → Real-time search functionality
